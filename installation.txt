Installationsanweisungen Microservices Workshop

Für den Microservices Workshop müssen folgende Tools auf den Computern der
Teilnehmer installiert werden:

1) Google Chrome

2) Git Client
    Download unter https://git-scm.com/download/win
    Bei der Installation alles bei Standardwerten belassen und den
    Installationsassistenten ausführen.

    Die erfolgreiche Installation kann über die Eingabeaufforderung mit folgendem
    Befehl verifiziert werden:

        git --version

    Die erwartete Ausgabe entspricht (ungefähr):

        git version 2.10.2.windows.1

3) Java 8 JDK
    Download unter http://www.oracle.com/technetwork/java/javase/downloads/jdk8-downloads-2133151.html
    (32 oder 64bit Version entsprechend dem Betriebssystem auswählen).

    Die erfolgreiche Installation kann über die Eingabeaufforderung mit folgendem
    Befehl verifiziert werden:

        java -version

    Die erwartete Ausgabe entspricht (ungefähr):

        java version "1.8.0_20"
        Java(TM) SE Runtime Environment (build 1.8.0_20-b26)
        Java HotSpot(TM) 64-Bit Server VM (build 25.20-b23, mixed mode)

4) IntelliJ Idea oder andere lieblings-IDE (Eclipse, o.ä.)

5) AWS Command Line Client
    Download der passenden Version:
        32 Bit: https://s3.amazonaws.com/aws-cli/AWSCLI32.msi
        64 Bit: https://s3.amazonaws.com/aws-cli/AWSCLI64.msi

    Bei der Installation alles bei Standardwerten belassen und den
    Installationsassistenten ausführen.

    Die erfolgreiche Installation kann über die Eingabeaufforderung mit folgendem
    Befehl verifiziert werden:

        aws --version

    Die erwartete Ausgabe entspricht (ungefähr):

        aws-cli/1.11.13 Python/2.7.9 Windows/7 botocore/1.4.70
